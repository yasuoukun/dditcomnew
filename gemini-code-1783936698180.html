💬LINE ID ฝ่ายบุคคล (สำหรับผู้ดูแลฟอร์ม)

แก้ไข LINE ID นี้ให้เป็น ID จริงของร้านก่อนเผยแพร่ลิงก์ให้ผู้สมัคร ระบบจะใช้อันนี้เปิดแอป LINE ทักแชตอัตโนมัติ

แบบฟอร์มนี้ทำงานทั้งหมดในเบราว์เซอร์ของท่าน ไม่มีการส่งข้อมูลไปยังเซิร์ฟเวอร์ภายนอกโดยอัตโนมัติ
