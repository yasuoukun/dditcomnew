📌 แบบฟอร์มนี้เป็นใบสมัครงานทั่วไปของบริษัท ใช้ได้กับทุกตำแหน่ง/ทุกฝ่ายงาน (เลือกฝ่ายที่สมัครได้ในข้อ 1) ระบบยังไม่ได้เชื่อมต่อฐานข้อมูลกลาง เมื่อกดปุ่ม "ตรวจทานและบันทึกใบสมัคร" ท่านจะได้หน้าสรุปสำหรับบันทึกเป็น PDF / สั่งพิมพ์ และดาวน์โหลดไฟล์ข้อมูล เพื่อส่งต่อให้บริษัท (ทาง LINE) — ก่อนเผยแพร่ลิงก์จริง อย่าลืมแก้ LINE ID ปลายทางในช่อง "LINE ID ฝ่ายบุคคล" ท้ายฟอร์มให้ตรงกับของบริษัท

ข้าพเจ้ารับรองว่าข้อมูลและเอกสารที่ให้ไว้ในใบสมัครนี้เป็นความจริงทุกประการ หากตรวจพบภายหลังว่าเป็นเท็จหรือปกปิดข้อเท็จจริง บริษัทมีสิทธิยุติกระบวนการรับสมัครหรือการจ้างงานได้ ข้าพเจ้ายินยอมให้บริษัทเก็บ ใช้ ตรวจสอบ และเปิดเผยข้อมูลส่วนบุคคลเท่าที่จำเป็นเพื่อการสรรหา ตรวจสอบประวัติ ติดต่อบุคคลอ้างอิง และดำเนินการด้านการจ้างงานตามกฎหมายและนโยบายของบริษัท

💬LINE ID ฝ่ายบุคคล (สำหรับผู้ดูแลฟอร์ม)

แก้ไข LINE ID นี้ให้เป็น ID จริงของบริษัทก่อนเผยแพร่ลิงก์ให้ผู้สมัคร ระบบจะใช้อันนี้เปิดแอป LINE ทักแชตอัตโนมัติ

แบบฟอร์มนี้ทำงานทั้งหมดในเบราว์เซอร์ของท่าน ไม่มีการส่งข้อมูลไปยังเซิร์ฟเวอร์ภายนอกโดยอัตโนมัติ
